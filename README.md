🧠 About Wellora

Wellora is an AI-powered healthcare assistant designed to provide intelligent insights based on user symptoms through both text and image inputs. It helps users understand possible health conditions, suggested medications, and wellness tips in a simple and accessible way.

The system supports multilingual interaction, allowing users to communicate in English, Hindi, and Telugu, making healthcare guidance more inclusive and user-friendly.

Wellora combines natural language processing and machine learning techniques to analyze symptoms, rank possible causes, and deliver structured responses that include:

Possible causes (ranked)
Recommended medicines
When to seek medical help
Personalized wellness tips

Additionally, Wellora integrates features inspired by Electronic Health Records (EHR), enabling better organization and tracking of user interactions.

⚠️ Wellora is intended for educational and informational purposes only and is not a substitute for professional medical advice.
